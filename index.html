Pagina Web de Natacion

Resumen del Proyecto

Fue un proyecto de la escuela, la página la desarrollamos mi equipo y yo. Te permite guardar, eliminar, actualizar los registros en una base de datos. Todos los formularios cuentan con sus respectivas validaciones, haciendo uso del AJAX para poder verificar lo que se pedía en dicho input.
Aplicamos todo lo aprendido durante el cuatrimestre, como fue el crear PDF's con base en una librería, hacer uso del Fullcalendar con todas sus funcionalidades que tiene ya implementadas, etc.
Fue de gran ayuda el realizar este proyecto para seguir practicando el desarrollo de páginas web.
